# Document Workflows

Step-by-step procedures for common document formatting tasks.

## Workflow 1: Format Raw Text to Professional Document

### Step 1: Initial Assessment
1. Read the entire document once
2. Identify document type (report, letter, academic, etc.)
3. Note existing formatting issues
4. Determine target formatting standard

### Step 2: Apply Document Structure
1. Apply appropriate template (if available)
2. Set document margins and page orientation
3. Add header and footer
4. Configure page numbers
5. Set line spacing and paragraph spacing

### Step 3: Apply Styles and Typography
1. Apply Normal style to all text (clear existing formatting)
2. Apply Heading styles to section headers
3. Set font size and style for body text
4. Configure title and subtitle formatting
5. Apply theme (if appropriate)

### Step 4: Format Lists and Tables
1. Convert text lists to proper bulleted/numbered lists
2. Apply list styles (consistent indentation)
3. Apply table styles to all tables
4. Configure table headers (repeat header rows)
5. Add table captions above tables

### Step 5: Format Figures and Images
1. Insert images at appropriate locations
2. Set image size (within page margins)
3. Configure text wrapping (Square or In Line)
4. Add figure captions below images
5. Ensure consistent sizing/alignment

### Step 6: Front Matter
1. Create title page (if required)
2. Add table of contents (Insert → Table of Contents)
3. Add list of figures/tables (if applicable)
4. Add executive summary or abstract (if needed)
5. Update all fields (F9)

### Step 7: Final Polish
1. Run spell check and grammar check
2. Check widows and orphans (single lines at page breaks)
3. Ensure section numbering is correct
4. Verify page breaks (no manual page breaks at page end)
5. Update all cross-references and TOC
6. Save with appropriate name in formatted-documents folder

## Workflow 2: Document Review and Revision

### Step 1: Enable Review Mode
1. Turn on Track Changes (Review → Track Changes)
2. Set display to "All Markup"
3. Configure markup display options:
   - Show: Comments, formatting, ink
   - Balloons: All revisions in margin
   - Paper orientation: Landscape for markup view

### Step 2: Content Review
1. Read for clarity and flow
2. Add comments for ambiguous sections
3. Suggest revisions via tracked changes
4. Check for consistency in terminology
5. Verify facts and figures

### Step 3: Structure Review
1. Check heading hierarchy (no skipped levels)
2. Verify section numbering sequence
3. Ensure transitions between sections
4. Check paragraph length and variety
5. Verify logical organization

### Step 4: Formatting Review
1. Check style consistency (same style for same elements)
2. Verify font sizes and styles
3. Check spacing (line, paragraph)
4. Verify table and figure numbering
5. Check page break locations

### Step 5: Language and Grammar
1. Run spelling and grammar check
2. Check for commonly confused words
3. Verify verb tense consistency
4. Check subject-verb agreement
5. Review passive vs. active voice

### Step 6: Finalize Revisions
1. Review all tracked changes
2. Accept or reject each change
3. Address all comments
4. Resolve conflicts
5. Turn off Track Changes
6. Accept all remaining changes
7. Save final version

## Workflow 3: Convert Plain Text to Structured Document

### Step 1: Import and Clean
1. Open or paste plain text
2. Remove extra line breaks (two or more)
3. Remove extra spaces
4. Replace multiple spaces with single space
5. Replace non-standard characters (smart quotes, em dashes)

### Step 6: Save and Organize
1. Save in formatted-documents folder
2. Use descriptive filename
3. Include version number if iterative
4. Note any limitations or manual fixes needed

## Workflow 4: Create Consistent Document Template

### Step 1: Define Template Requirements
1. Document types that will use template
2. Required styles and formatting
3. Standard sections (headers, footers, cover pages)
4. Branding elements (logo, colors, fonts)
5. Any automation needs (macros, fields)

### Step 2: Set Up Document Structure
1. Create new document
2. Set page setup (margins, orientation, size)
3. Add headers and footers
4. Add page numbers
5. Set up multiple sections if needed (different first page, landscape pages)

### Step 3: Create Styles
1. Modify or create styles for:
   - Title (24-28pt, bold)
   - Subtitle (18pt, bold)
   - Heading 1-9 (hierarchical)
   - Normal (11-12pt, justified)
   - Quote (italic, indented)
   - Caption (9-10pt, italic)
   - List Paragraph (for lists)
2. Set font, size, spacing for each
3. Assign keyboard shortcuts if frequently used
4. Add styles to Quick Styles gallery

### Step 4: Add Building Blocks
1. Cover pages
2. Sample headers/footers
3. Sample tables (formatted)
4. Text boxes (formatted)
5. Watermarks (confidential, draft, etc.)

### Step 5: Add Automation
1. Create macros for common tasks
2. Add document properties (Title, Author, etc.)
3. Insert fields for automatic dates, filenames
4. Create custom toolbars or ribbon tabs (advanced)

### Step 6: Test and Refine
1. Create sample document using template
2. Test all styles and automation
3. Get user feedback
4. Refine based on testing
5. Document template usage instructions

### Step 7: Save and Distribute
1. File → Save As → Word Template (.dotx)
2. Save to Custom Office Templates folder
3. Create template usage guide
4. Distribute to users
5. Provide training if needed

## Workflow 5: Format Complex Document with Multiple Sections

### Step 1: Plan Document Structure
1. Outline all sections and subsections
2. Identify special formatting needs:
   - Landscape pages (tables, figures)
   - Different headers/footers (chapters, appendixes)
   - Multiple column layouts
   - Special page numbering (i, ii, iii vs 1, 2, 3)

### Step 2: Create Section Structure
1. Insert section breaks (NOT page breaks):
   - Next page: For chapters
   - Continuous: For column changes
   - Even/Odd page: For book-style layout
2. Configure each section independently:
   - Page orientation
   - Margins
   - Headers and footers
   - Page numbering format
   - Columns

### Step 3: Configure Headers and Footers
1. Double-click header/footer area
2. Check "Different First Page" for title pages
3. Check "Different Odd and Even Pages" for books
4. Link to previous (or unlink) as needed
5. Add appropriate content:
   - Section titles in headers
   - Page numbers in footers
   - Company logo or document info

### Step 4: Configure Page Numbering
1. Insert page numbers in appropriate location
2. Set format for each section:
   - Roman numerals (i, ii, iii) for front matter
   - Arabic numerals (1, 2, 3) for main content
   - Restart numbering at 1 for each chapter (if needed)
3. Update page numbers in TOC and cross-references

### Step 5: Format Tables and Figures
1. Create tables in appropriate sections
2. Use landscape orientation for wide tables
3. Apply consistent table styles
4. Add table captions
5. Insert table of tables (if many tables)
6. Format figures consistently
7. Add figure captions
8. Insert table of figures (if many figures)

### Step 6: Generate Front Matter
1. Title page (no page number)
2. Copyright page (if applicable)
3. Abstract or executive summary
4. Table of contents (Insert → Table of Contents)
5. List of tables (Insert → Table of Tables)
6. List of figures (Insert → Table of Figures)
7. Update all tables/figures (F9)

### Step 7: Quality Check
1. Check section boundaries (view in Draft mode)
2. Verify headers/footers change appropriately
3. Check page numbering sequence
4. Ensure all tables/figures are numbered
5. Verify all cross-references work
6. Update all fields (F9)
7. Test print preview

## Workflow 6: Merge Multiple Documents

### Step 1: Prepare Source Documents
1. Ensure all documents use same template/styles
2. Remove page breaks from end of documents
3. Note any special formatting in source documents
4. Backup originals before merging

### Step 2: Insert Documents
1. Open main document (or create new)
2. Position cursor where insertion will occur
3. Insert → Object → Text from File
4. Select document to insert
5. Repeat for all documents to merge

### Step 3: Standardize Formatting
1. Select all (Ctrl+A)
2. Apply appropriate styles to headings
3. Apply Normal style to body text
4. Fix any manual formatting
5. Update numbering (if sequential)

### Step 4: Update Front Matter
1. Update table of contents
2. Update list of tables/figures
3. Update cross-references
4. Update page numbers
5. Verify section numbering

### Step 5: Final Review
1. Check for consistent formatting
2. Verify page numbering is continuous
3. Check headings and subheadings
4. Verify tables and figures are renumbered
5. Check for orphaned headings at bottom of pages
6. Update all fields (F9)

## Common Issues and Solutions

### Issue: Styles Not Applying
**Solution**:
1. Clear direct formatting (Clear All)
2. Ensure style exists in document
3. Check "Based on" style isn't overriding
4. Reboot if document is corrupt

### Issue: Page Numbers Wrong
**Solution**:
1. Check section breaks (not page breaks)
2. Verify "Link to Previous" setting
3. Check page number format (i, ii vs 1, 2)
4. Restart numbering at section break if needed

### Issue: TOC Not Updating
**Solution**:
1. Right-click TOC → Update Field
2. Update entire table (not just page numbers)
3. Check heading styles are applied
4. Verify TOC includes correct levels

### Issue: Tables Split Across Pages
**Solution**:
1. Select table → Table Tools → Layout
2. Properties → Row → "Allow row to break across pages"
3. Or: Keep rows together
4. Adjust column widths to fit

### Issue: Images Moving Unexpectedly
**Solution**:
1. Change wrapping to "In Line with Text" for stable positioning
2. Or: Use "Square" with anchor lock
3. Set absolute position
4. Group image with caption (advanced)
