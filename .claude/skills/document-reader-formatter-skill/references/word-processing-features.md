# Word Processing Features Reference

Comprehensive guide to advanced Word document processing features.

## Document Navigation

### Navigation Pane
- **Purpose**: Browse by headings, pages, or search results
- **Enable**: View tab → Navigation Pane
- **Use Cases**: Quick jumping between sections, restructuring documents
- **Tip**: Drag headings in Navigation Pane to reorganize

### Cross-References
- **Insert**: Insert → Cross-reference
- **Types**: Headings, figures, tables, numbered items
- **Update**: F9 (update field) or right-click → Update Field
- **Best Practice**: Use for all references to other document elements

### Bookmarks
- **Insert**: Insert → Bookmark
- **Purpose**: Mark locations for quick navigation
- **Use Cases**: Jump to specific sections, create hyperlinks
- **Naming**: No spaces, descriptive (e.g., "Chapter1_Intro")

### Hyperlinks
- **Insert**: Ctrl+K or Insert → Link
- **Types**: Web URLs, document bookmarks, email addresses
- **Display text**: Different from URL
- **Remove**: Ctrl+Shift+F9 (convert to text)

## Table Management

### Table Creation Methods

1. **Insert Table Dialog**: Specify rows/columns
2. **Draw Table**: Freeform drawing
3. **Convert Text**: Select text → Insert → Table → Convert Text to Table
4. **Excel Spreadsheet**: Insert → Table → Excel Spreadsheet

### Table Styles Application
- **Apply**: Table Tools → Design → Table Styles
- **Customize**: Modify Table Style
- **Clear**: Table Tools → Design → Table Styles → Clear
- **Best Practice**: Apply styles, not manual formatting

### Advanced Table Features

**Repeat Header Rows**:
- Table Tools → Layout → Repeat Header Rows
- Purpose: Headers appear on each page
- Automatically adjusts for page breaks

**Formulas in Tables**:
- Layout → Formula
- Common functions: SUM, AVERAGE, PRODUCT, MIN, MAX
- Cell references: LEFT, RIGHT, ABOVE, BELOW
- Update: F9 or right-click → Update Field

**Table Sorting**:
- Layout → Sort
- Sort by: Up to 3 levels
- Type: Text, number, date
- Header row: Check if has headers

**Table Conversion**:
- Table to text: Layout → Convert to Text
- Text to table: Select text → Insert → Table
- Separators: Paragraphs, tabs, commas, other

### Table Layout Management

**Column Width**:
- AutoFit: AutoFit Window, AutoFit Contents
- Specific width: Table Tools → Layout → Cell Size → Width
- Distribute columns: Even spacing
- Ruler: Drag column boundaries

**Row Height**:
- At least: Minimum height
- Exactly: Fixed height (can cause text overflow)
- Distribute rows: Even spacing

**Merge/Split Cells**:
- Merge: Select cells → Layout → Merge Cells
- Split: Layout → Split Cells (specify rows/columns)
- Use case: Complex table headers, spanning columns

**Text Direction**:
- Layout → Text Direction
- Options: Horizontal, vertical, rotate 90°, 270°
- Use case: Fitting narrow columns, Asian languages

## Styles & Themes

### Style Types

**Character Styles**:
- Apply to selected text only
- Examples: Emphasis, Strong, Book Title
- Font: Paragraph mark not affected

**Paragraph Styles**:
- Apply to entire paragraph
- Examples: Normal, Heading 1-9, Quote
- Includes font, spacing, alignment, numbering

**Linked Styles**:
- Can be paragraph or character
- Example: Heading styles (linked by default)
- Double-click character icon in Styles pane

### Style Management

**Create New Style**:
- Styles pane → New Style
- Based on: Usually Normal
- Style for following paragraph: Auto or specific
- Add to Quick Styles: For easy access

**Modify Style**:
- Right-click style → Modify
- Changes update all instances
- "Automatically update": Only for current document (avoid)

**Style Organizer**:
- Manage styles in template or document
- Copy styles between documents
- Delete unused styles
- Rename styles

**Style Inspector**:
- Shows direct formatting vs. style formatting
- Reveal formatting: Shift+F1
- Clear formatting: Clear All button

### Themes

**Document Themes**:
- Design → Themes
- Affects: Colors, fonts, effects
- Purpose: Consistent appearance across documents

**Theme Colors**:
- Design → Colors
- Sets: Accent colors, hyperlink colors
- Customizable: Create new color sets

**Theme Fonts**:
- Design → Fonts
- Heading font + Body font
- Example: Calibri-Calibri (heading, body)

**Theme Effects**:
- Design → Effects
- Applies to: Shapes, SmartArt, objects
- Not: Text formatting

## Graphics & Visual Elements

### Images

**Insert Options**:
- Insert → Pictures (from file)
- Insert → Screenshot (screen clipping)
- Insert → Online Pictures (Bing, OneDrive)
- Copy/paste: From web or other documents

**Image Formatting**:
- **Size**: Drag corners (maintain aspect ratio)
- **Crop**: Picture Tools → Crop
- **Wrap Text**: In Line with Text, Square, Tight, etc.
- **Position**: More layout options

**Picture Styles**:
- Picture Tools → Picture Styles
- Effects: Shadow, reflection, glow, bevel
- Borders: Various weights and colors
- Reset: Reset Picture button

**Image Corrections**:
- Corrections: Brightness, contrast, sharpen/soften
- Color: Saturation, tone, recolor
- Artistic effects: Pencil sketch, paint strokes, etc.

**Alt Text**:
- Right-click → Edit Alt Text
- Required for: Accessibility compliance
- Describe: Image content and context

### Shapes & SmartArt

**Shapes**:
- Insert → Shapes
- Types: Lines, rectangles, basic shapes, arrows, flowchart, callouts
- Format: Shape Styles, fill, outline, effects
- Text: Add text, format separately

**SmartArt**:
- Insert → SmartArt
- Categories: List, Process, Cycle, Hierarchy, Relationship, Matrix, Pyramid, Picture
- Convert: Convert existing shapes to SmartArt
- Design pane: Change layout, colors, styles

**Text Boxes**:
- Insert → Text Box
- Types: Built-in styles, draw text box
- Format: Shape fill, outline, effects
- Link: Link text boxes for overflow

### Charts

**Insert Chart**:
- Insert → Chart
- Types: Column, line, pie, bar, area, scatter, stock
- Data: Edit in Excel window
- Design: Chart styles, color schemes

**Chart Elements**:
- Axes: Primary, secondary, axis titles
- Gridlines: Major, minor
- Labels: Data labels, axis labels
- Legend: Position, formatting
- Trendlines: Linear, exponential, etc.

## Proofreading Tools

### Spelling & Grammar
- **Location**: Review tab → Spelling & Grammar (F7)
- **Settings**: File → Options → Proofing
- **Custom dictionary**: Add technical terms
- **Check**: Context menu for suggestions

### Thesaurus
- **Access**: Review tab → Thesaurus (Shift+F7)
- **Right-click**: Synonyms option
- **Language**: Multiple languages supported
- **Integration**: Replace word directly from pane

### Word Count
- **Location**: Review → Word Count
- **Statistics**: Words, characters, paragraphs, lines
- **Selection**: Count selected text only
- **Goal**: Set writing goals

### AutoCorrect
- **Options**: File → Options → Proofing → AutoCorrect
- **Features**: Replace typos, capitalize first letter, fix capitalization
- **Custom**: Add your own corrections
- **Math**: AutoCorrect for math symbols

### Read Aloud
- **Access**: Review → Read Aloud
- **Voice**: System text-to-speech
- **Speed**: Adjustable
- **Use case**: Proofreading by listening

## Track Changes & Comments

### Track Changes
- **Enable**: Review → Track Changes
- **Display**: All Markup, Simple Markup, No Markup
- **Settings**: Change font, color, position of markup
- **Accept/Reject**: Review → Accept/Reject buttons

### Comments
- **Insert**: Review → New Comment (Ctrl+Alt+M)
- **Reply**: Reply to existing comments
- **Delete**: Delete button or context menu
- **Resolve**: Mark as resolved (hides but keeps)

### Compare Documents
- **Compare**: Review → Compare
- **Options**: Show source, revised, or combined
- **Changes**: Tracked as revisions
- **Merge**: Combine multiple revisions

## Automation Features

### Macros
- **Record**: View → Macros → Record Macro
- **Store**: Document or Normal.dotm (global)
- **Run**: View → Macros → View Macros → Run
- **Security**: Enable macros in Trust Center

### Field Codes
- **Insert**: Insert → Quick Parts → Field
- **Common fields**: Date, Page, TOC, Index, FileName
- **Update**: F9 or right-click → Update Field
- **Toggle codes**: Alt+F9

### Building Blocks
- **Access**: Insert → Quick Parts → Building Block Organizer
- **Types**: Cover pages, headers, footers, text boxes, watermarks
- **Create**: Select text → Insert → Quick Parts → Save Selection
- **Insert**: Reusable in any document

### Templates
- **Create**: File → Save As → Word Template (.dotx)
- **Location**: Custom Office Templates folder
- **Use**: File → New → Personal
- **Contain**: Styles, macros, building blocks, default text

## Document Protection

### Restrict Editing
- **Location**: Review → Restrict Editing
- **Options**: No changes (read-only), comments, filling in forms
- **Exceptions**: Allow specific users to edit
- **Password**: Optional password to stop protection

### Mark as Final
- **Purpose**: Prevent accidental edits
- **Status**: Read-only recommendation
- **Remove**: File → Info → Protect Document → Mark as Final

### Digital Signatures
- **Add**: Add visible or invisible signature line
- **Purpose**: Verify document authenticity
- **Certificate**: Required from certificate authority

### Password Protection
- **Open password**: Required to open document
- **Modify password**: Required to edit
- **Set**: File → Info → Protect Document → Encrypt with Password
- **Warning**: Cannot recover lost passwords
