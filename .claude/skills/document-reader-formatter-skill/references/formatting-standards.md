# Professional Document Formatting Standards

Complete reference for formatting business, academic, and enterprise documents to professional standards.

## Document Type Standards

### Business Documents

**Reports**:
- Font: 11-12pt (Calibri, Arial, or Garamond)
- Margins: 1" all sides (0.5" for header/footer)
- Line spacing: 1.15 or 1.5
- Paragraph spacing: 6pt after paragraphs
- Headers: 16-18pt bold, numbered sections
- Page numbers: Bottom center or bottom right

**Letters**:
- Font: 11-12pt professional font
- Block format: Left-aligned all text
- Date: 2-3 lines below letterhead
- Inside address: Below date, left-aligned
- Salutation: "Dear [Name]:"
- Body: 3-4 paragraphs max
- Closing: "Sincerely," or "Best regards,"
- Signature: 4 lines below closing

**Memorandums**:
- Format: MEMORANDUM centered at top
- To/From/Date/Subject: Bold labels, left-aligned
- Font: 11-12pt
- Single spacing within paragraphs
- Double spacing between paragraphs

### Academic Documents

**Research Papers** (APA Style):
- Font: 12pt Times New Roman
- Margins: 1" all sides
- Double spacing throughout
- Page numbers: Top right
- Running head: Top left (50 characters max)
- Headings: Level 1 (centered bold), Level 2 (left-aligned bold), Level 3 (bold italic)
- References: Separate page, hanging indent

**Essays** (MLA Style):
- Font: 12pt Times New Roman
- Margins: 1" all sides
- Double spacing
- No title page (information on first page)
- Page numbers: Top right, last name + number
- Citations: Parenthetical (Author Page)
- Works Cited: Separate page, alphabetical

### Enterprise Documents

**Proposals**:
- Executive summary: 1-2 pages
- Table of contents: Auto-generated
- Section numbering: 1, 1.1, 1.1.1
- Headers/footers: Company logo, confidentiality notice
- Appendix: Supporting documents
- Font: 10-12pt (tables can use 9pt)

**Technical Documentation**:
- Consistent terminology throughout
- Version control: Footer with version, date, author
- Diagrams: Numbered and captioned
- Code blocks: Monospace font, light gray background
- Cross-references: Hyperlinked when digital

## Typography Standards

### Font Selection Guidelines

**Serif Fonts** (Formal, academic, print):
- Times New Roman (11-12pt)
- Garamond (11-12pt)
- Georgia (11pt)
- Book Antiqua (11pt)

**Sans-Serif Fonts** (Modern, digital, presentations):
- Calibri (11pt)
- Arial (10-11pt)
- Helvetica (10-11pt)
- Verdana (10pt)

**Monospace Fonts** (Code, data):
- Consolas or Courier New (9-10pt)
- For code blocks only

### Font Hierarchy

| Element | Size | Style | Example |
|---------|------|-------|---------|
| Title | 24-28pt | Bold | Report Title |
| Section Headers | 16-18pt | Bold | 1. Introduction |
| Subsection Headers | 14pt | Bold | 1.1 Background |
| Body Text | 11-12pt | Regular | Paragraph text |
| Captions | 9-10pt | Italic | Figure 1. Graph |
| Footnotes | 8-9pt | Regular | Note: Data... |

## Color Standards

### Professional Color Palettes

**Business Professional**:
- Primary: Navy blue (#000080)
- Accent: Dark gray (#333333)
- Highlight: Light blue (#ADD8E6)
- Avoid: Red (indicates errors)

**Academic**:
- Primary: Black (#000000)
- Accent: Dark blue (#00008B)
- Highlight: Yellow (#FFFF00) for highlighting
- Subheading: Dark gray (#4F4F4F)

**Modern/Creative**:
- Primary: Charcoal (#36454F)
- Accent: Teal (#008080)
- Highlight: Light gray (#D3D3D3)
- Emphasis: Maroon (#800000)

### Color Usage Rules
- Never use light text on light backgrounds
- Maintain 4.5:1 contrast ratio (WCAG AA standard)
- Use color purposefully, not decoratively
- Test grayscale printing compatibility

## Spacing & Layout Standards

### Standard Margins
- **Normal**: 1" all sides
- **Wide**: 1.5" left/right (for binding)
- **Narrow**: 0.5" all sides (drafts only)
- **Mirror**: 1" inside, 0.75" outside (bound documents)

### Line Spacing
- **Single**: 1.0 (tables, captions)
- **1.15**: Standard business documents
- **1.5**: Academic drafts, legal documents
- **Double**: 2.0 (final academic papers)

### Paragraph Spacing
- **Before**: 0pt (usually)
- **After**: 6pt (business), 10pt (academic sections)
- **First line indent**: 0.5" (academic), none (business)

### Tab Stops
- **Decimal**: For numbers with decimals
- **Left**: Standard bullets (0.5", 1.0", 1.5")
- **Right**: For dates/alignments
- **Center**: For headers

## Header & Footer Standards

### Headers
- **Position**: 0.5" from top
- **Content**: Document title or chapter name
- **Font**: 9-10pt, italic or regular
- **Alignment**: Center or left
- **Different first page**: Yes (title pages)

### Footers
- **Position**: 0.5" from bottom
- **Content**: Page numbers, file path, date
- **Font**: 9-10pt
- **Page number formats**:
  - "Page X of Y"
  - "X" (simple number)
  - "Chapter X - Page Y"

## Numbering & Bullets Standards

### Outline Numbering
```
1. First Level
   1.1 Second Level
      1.1.1 Third Level
         1.1.1.1 Fourth Level
```

### Bullet Hierarchy
```
Level 1: ● Solid circle
Level 2: ○ Open circle
Level 3: ▪ Square
Level 4: - Dash
```

### Legal Numbering
```
1. Section one
   a. Subsection
      i. Item
         (1) Sub-item
```

## Table & Figure Standards

### Table Formatting
- **Alignment**: Centered on page
- **Borders**: Top/bottom heavy, internal light
- **Header row**: Bold, shaded background
- **Font**: 1-2pt smaller than body
- **Spacing**: 0pt before/after within cells
- **Caption**: Above table, "Table X: Description"

### Figure/Image Formatting
- **Alignment**: Centered or with text wrap
- **Caption**: Below image, "Figure X: Description"
- **Size**: No larger than page margins
- **Resolution**: Minimum 300 DPI for print
- **Borders**: Simple line if needed

### Table Styles
| Style Type | Usage | Characteristics |
|------------|-------|------------------|
| Simple | Data lists | Minimal borders |
| Grid | Detailed data | All borders visible |
| Light | Professional | Header shading only |
| Medium | Reports | Alternating row colors |
| Dark | Emphasis | Heavy header shading |

## Section Breaks & Pagination

### Section Break Types
- **Next page**: New chapter/section
- **Continuous**: Column changes within page
- **Even page**: Start chapter on left page
- **Odd page**: Start chapter on right page

### Page Breaks
- Use instead of multiple Enter keys
- Before headings at bottom of page
- Keep with next: For headings
- Keep lines together: For paragraphs

## Consistency Standards

### Style Enforcement
- **Direct formatting**: NEVER (use styles instead)
- **Style names**: Descriptive (Heading 1, Body Text, Caption)
- **Based on**: Cascade from parent styles
- **Update**: Modify style, not individual instances

### Template Structure
1. **Normal style**: Base for all styles
2. **Heading styles**: 1-9 levels
3. **Special styles**: Title, Subtitle, Quote, Caption
4. **List styles**: Bulleted, Numbered
5. **Table styles**: Consistent formatting

## Quality Checklist

Before finalizing any document:

- [ ] Font sizes consistent within element types
- [ ] Line spacing uniform throughout
- [ ] Margins correct for document type
- [ ] Page numbers present and correct
- [ ] Headers/footers appropriate
- [ ] Tables have captions and consistent borders
- [ ] Figures numbered and captioned
- [ ] Cross-references work (if digital)
- [ ] Section numbering sequential
- [ ] No widows/orphans (single lines at page breaks)
- [ ] Table of contents current (if applicable)
- [ ] Spell check completed
- [ ] Grammar check completed
